# Bug Scratch Pad

Quick-capture log for bugs found outside GitHub. Add entries here freely — periodically ask Claude to review and convert to GitHub Issues.

**Format per entry:**
```
### [Component] Short description
Date: YYYY-MM-DD
Route: /path/to/page
Severity: Critical | High | Medium | Low
Steps: (brief)
Expected: (what should happen)
Actual: (what happened)
Notes: (any extra context)
Status: Open | Converted to #<issue> | Fixed
```
